# TravGPT

**TravGPT** is a Streamlit application designed to answer questions using context from PDF or text files. It integrates Cohere for generating answers and uses FAISS for efficient text search.

## Features

- Extracts and processes text from PDF or text files.
- Splits text into contextually relevant chunks.
- Utilizes FAISS for vector-based search and retrieval.
- Generates answers with Cohere based on the provided context.

## Setup

1. Clone the repository and navigate to the project directory.
2. Set up a virtual environment and install the required dependencies.
3. Configure your API keys and file paths in the application settings.

## Usage

- Launch the application using Streamlit.
- Access the application through your web browser.
- Input your question and get context-based answers.
